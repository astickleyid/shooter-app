About

What is VOID RIFT?

VOID RIFT is a twin-stick space shooter built as a complete game product — entirely in vanilla JavaScript using the Canvas API, with no game engine. Players choose from a roster of ships with distinct stats, survive escalating enemy waves, earn in-game currency, and spend it in a persistent upgrade shop between sessions.

The game combines classic arcade action with modern mobile game design: a roguelite wave-upgrade system delivers a new power each wave, a kill combo multiplier rewards aggressive play, and an adaptive difficulty curve keeps sessions challenging at every skill level. It's free to start and rewards long-term play with a global leaderboard, a mission system, daily challenge runs, and an achievement system with 10 unlockable accomplishments.

Long description

VOID RIFT is a fast-paced twin-stick space shooter built for mobile — free to play, no account required, and designed to feel just as good on a small screen as it does in a browser.

Choose from 8 ships, each with different speed, armor, firepower, and a unique special ability. Fight through escalating waves of enemy types — carriers that deploy drones, phantoms that cloak, berserkers that charge, snipers that fire from range, and boss waves that push your skills to the limit.

Between runs, spend earned credits in the upgrade shop to permanently improve your stats. Each wave also offers a roguelite power boost — pick one of three options to adapt your build mid-run. Kill combos multiply your score. Power-ups drop in the heat of combat.

Features:
• 8 ships with distinct weapons and special abilities
• Persistent upgrade shop between sessions
• Roguelite wave upgrades — new power every wave
• 8 enemy types including bosses
• Daily Challenge — new seeded run every day
• Mission system with objectives and rewards
• 10 achievements to unlock
• Global leaderboard
• No ads during gameplay — rewarded ads only on the death screen (optional)
• Remove Ads available as a one-time purchase

Developer

About the Developer

VOID RIFT is an independent game built by Austin Stickley, a solo product engineer and game developer based in the United States. The game is written entirely in vanilla JavaScript using the Canvas API — no game engine — and packaged for iOS and Android via Capacitor. It is the most technically self-contained project in Austin's portfolio, with every system — physics, save state, leaderboard, social, adaptive difficulty, audio — built by hand in a single codebase.
