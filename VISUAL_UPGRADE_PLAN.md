### How to Know It Worked:
1. **Visual Density**: Screen should feel alive with particles, effects, animations
2. **Clarity**: Despite more effects, information is clearer

4. **Polish**: No rough edges, smooth everywhere
5. **WOW Moments**: Players say "that looks amazing" not "that works"

### Specific Goals:
- 3x more particle effects on screen
- 5+ animation states per UI element
- Sub-100ms response to all interactions
- Zero visual bugs or glitches
- Consistent design language throughout
